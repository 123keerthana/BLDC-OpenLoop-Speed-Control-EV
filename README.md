# Open Loop Speed Control of BLDC Motor for EV Applications

## Overview
Developed and simulated a complete BLDC motor drive system for Electric Vehicle applications using MATLAB/Simulink. The project implements 120° six-step inverter commutation with Hall sensor based control and PI speed regulation.

## Technical Contributions
- Mathematical modeling of BLDC motor (electrical & mechanical)
- Trapezoidal Back-EMF generation
- Hall sensor decoding logic
- 3-phase VSI inverter modeling
- Open loop and PI closed loop speed control
- Load disturbance analysis
- Hardware validation using dSPACE 1104 and OPAL-RT

## Tools & Skills
- MATLAB / Simulink  
- Power Electronics  
- Motor Control – BLDC  
- Embedded Interface – dSPACE  
- EV Drive Systems

## Results Achieved
- Stable speed tracking under load  
- Correct six-step commutation  
- Trapezoidal back-EMF verified  
- Hardware: ~32V RMS, ~3.5A current
